Research Projects

Resource-Constrained Computer Vision
Low-power vision pipelines for edge and embedded platforms
Edge AI Embedded Vision Efficiency

Developed lightweight computer vision and machine learning solutions for agricultural, industrial, and public safety applications, including peanut maturity classification using hyperspectral feature extraction, chlorophyll estimation in winter wheat, smartphone-based color matching for smart textiles, keyway angle detection in manufacturing systems, and vision-based social distancing measurement for autonomous vehicles.

In-Memory / In-Sensor Computer Vision
Compute near data for ultra-low-power sensing and inference
In-Sensor In-Memory Mixed-Signal

Developed a post–von Neumann visual system integrating sensing, memory, and computation using memristive crossbar arrays. By pushing dot-product operations and classification directly into hardware with analog WTA and mixed-signal peripherals, the system enables low-power, real-time edge vision.

Hardware-Aware Learning / Neuromorphic Computing
Algorithms co-designed with circuits and devices for efficiency
SNNs Neuromorphic Co-Design

Designed energy-efficient spiking neural networks using current-controlled memristive synapses and hardware-aware mapping. The framework accounts for device variability, ADC limits, and non-idealities, while incorporating sleep-inspired reservoir computing for adaptive, memory-constrained learning.
